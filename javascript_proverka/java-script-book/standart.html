Как получить кредит?

  1. Подготовьте необходимые документы (список документов здесь);
  2. Заполните форму заявления (скачать);
  3. Обратитесь в любое отделение банка с заполненным заявлением и пакетом документов;
  4. Заявление будет рассмотрено в течение 5 рабочих дней, после чего вам сообщат о решении Банка*
* - срок рассмотрения кредита может быть увеличен по усмотрению банка
